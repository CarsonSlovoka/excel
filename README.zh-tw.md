## 專看簡介

你們看到的這個庫名叫excel，當初的想法只是覺得這個名稱可能比較多人知道，另外想達到類似的功能(但僅侷限於檢視部分)

excel的功能其實已經很完善，但用COM控制很麻煩，而且不能很便捷的瀏覽到本機端的資料，需要強制把圖片篩進去，

而本專案的特色可以靠純餵入CSV檔案和指定static資料夾就能以CSV的方式瀏覽到圖片。
